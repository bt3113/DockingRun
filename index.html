Unofficial fan-made browser game

Docking Run

Fly near the singularity, recover mission data, survive time dilation, and complete the final docking sequence.

Local Best0

Controls

Desktop: W/↑/Space burn, A/← and D/→ rotate, E/Enter dock, R restart.

No official movie assets are used. This is an open-source fan project built around original visuals, physics-inspired mechanics, and score sharing.
